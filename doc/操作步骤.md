#### 初始化项目

winpty vue.cmd create vue-mobile-template

#### 基础配置路由

#### 全局使用 vant

yarn add vant

使用全局引入，main.js 引入
```
import Vant from 'vant';
import 'vant/lib/index.css';

Vue.use(Vant);
```

#### 自动添加前缀

```
yarn add --dev autoprefixer

如果有报错，版本太高，需要下载

yarn add --dev  autoprefixer@8.0.0
```

#### 使用 viewport 自适应

安装插件

```
yarn add postcss-px-to-viewport -D
```

根目录新建 .postcssrc.js 配置文件
